# Workspace Boundary Notice

This repository is a **public workspace**.

Rules:
- Work only inside this repository unless the user explicitly names a different workspace.
- Do **not** touch any `private`, `pvt`, or similarly named workspace, container, volume, or network by assumption.
- Do **not** connect this repository to any private or pvt infrastructure unless the user explicitly asks for that exact action.
- Do **not** reuse secrets, env files, docker resources, or network settings from a private workspace here.
- If a task would cross the public/private boundary, stop and ask for explicit permission.

This notice applies to all agents and LLMs operating in this repository.
